{{< callout type="verdict" title="Board Customization Basics" >}}
Added in v1.2.0, Board Customization lets you choose from **21 different boards** (4 default + 17 unlockable). Each board is purely cosmetic. No gameplay advantage. But the unlock conditions range from trivial to genuinely challenging. This guide covers every board, how to unlock it, and the fastest unlock path.
{{< /callout >}}

{{< diagram src="board-unlock-tree.svg" alt="Board unlock progression from easy to hard" caption="21 boards grouped by unlock difficulty. Easy = Green, Medium = Orange, Hard = Red, Hidden = Purple." >}}

Board customization unlocks in the main menu after your first win. Access it from the "Customize" tab.

## Default Boards (4)

These are available from the start:

| Board | Description |
|-------|-------------|
| Classic | The standard board you start with |
| Dark Mode | Dark theme variant |
| Marble | Clean marble texture |
| Wood | Classic wood grain |

## Unlockable Boards (17)

### Easy Unlocks (Win 1-3 Runs)

| Board | Unlock Condition | Tips |
|-------|-----------------|------|
| Ruby | Win any 1 run | Automatic after first clear |
| Sapphire | Win any 3 runs | Keep grinding |
| Emerald | Complete the tutorial | Takes 5 minutes |
| Golden | Collect 30 items in Collection | Play a few runs to fill your Collection |
| Ice | Beat Boss 1 without using a Gambit | Use basic chess moves only. Easy on Rook difficulty. |

### Medium Unlocks (Requires Specific Conditions)

| Board | Unlock Condition | Tips |
|-------|-----------------|------|
| Lava | Capture 50 pieces in a single run | Focus on economy and board control. Queen builds excel here. |
| Crystal | Collect 5 Gambits in a single run | Visit the Gachapon. Pick Gambit-giving nodes. |
| Shadow | Win a run without losing any pieces | Play defensively. Use PROTECT tiles. PAWN-heavy boards recommended. |
| Neon | Sell 20 pieces in a single run | Aggressive pruning strategy. Combine with economy Gambits. |
| Galaxy | Win a run on BISHOP difficulty or higher | Medium difficulty unlock. Requires you to progress past Easy. |
| Lunar | Use 3 different Royal Piece types in one run | Queen + Rook + Bishop combo. See [Build Guide](/build-guide/). |

### Hard Unlocks

| Board | Unlock Condition | Tips |
|-------|-----------------|------|
| Sunset | Win 10 runs total | Grind. Play fast runs on Rook difficulty. |
| Void | Beat the Final Boss | Requires a complete run. See [Final Boss Guide](/final-boss-hidden-phase/). |
| Phantom | Unlock all other boards first | The "completionist" board. This is your last one. |

### Hidden Boards

| Board | Unlock Condition | Note |
|-------|-----------------|------|
| Lighthouse | Start a game with only 1 PAWN on the board | Easter egg. Reference to the upcoming game "Lighthouse." |
| Beta | Have a pre-release save file | Only for demo/playtest players |
| Stray Fawn | Win a run on KING difficulty | Studio tribute board |
| Arcade | Win a run in under 10 minutes | Speedrun board. Focus on fast captures and economy skip. |

## Fastest Unlock Path

If you want every board ASAP:

1. **Day 1**: Complete tutorial to Emerald. Win 1 run to Ruby. Beat Boss 1 without Gambits to Ice.
2. **Day 2**: Win 3 runs total to Sapphire. Collect 30 items to Golden.
3. **Day 3**: Focus on conditional boards (Lava/Crystal/Shadow/Neon). One optimized run per board.
4. **Day 4-5**: BISHOP+ runs for Galaxy/Lunar. Push to Final Boss for Void.
5. **Day 6**: KING difficulty run to Stray Fawn. Speedrun to Arcade.
6. **Last**: Win 10 total to Sunset. Then Phantom unlocks automatically.
