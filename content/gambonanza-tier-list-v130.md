{{< callout type="verdict" title="v1.3.0 Tier List at a Glance" >}}
This tier list replaces the v1.1.0 version. **Templar drops from S to B-tier** (nerfed from 1/1 to 1/3). Six Gambits enter S and A-tier thanks to guaranteed triggers. Missigno's debuts in S-tier for lategame pivot potential. If you're still playing the v1.1.0 meta, your tier assumptions are wrong.
{{< /callout >}}

## S-Tier (Game-Warping)

| Gambit | Why | Best For |
|--------|-----|----------|
| **Missigno's Gambit** ★ NEW | Sell to randomize all pieces. Lategame hail-mary. | Stage 4+ pivots |
| **Catapult's Gambit** | ROOK capture launches diagonally. Board control king. | Any build |
| **Whip's Gambit** ↑ buffed to 1/1 | Guaranteed trigger now. Elite economy. | Economy builds |
| **Dead or Alive's Gambit** ↑ buffed to 1/1 | Guaranteed payout on capture. S-tier now. | Aggressive builds |
| **Graal's Gambit** ★ NEW | Free Blessing Tile on every Gambit pickup. | Tile-based strats |
| **Bunker's Gambit** ↑ reworked | Global +$1 per PROTECT. Consistent income all run. | Protect builds |

## A-Tier (Strong)

| Gambit | Why |
|--------|-----|
| **Show Jumping** ↑ 1/1 | Guaranteed jump. Knight synergy peak. |
| **Pendant** ↑ 1/1 | Guaranteed pendant. |
| **Chamberlain** ↑ 1/1 | Consistent piece generation now. |
| **Miracle** ↑ 1/1 | Took 3 patches to get here, but finally reliable. |
| **Princess's Gambit** | Bless ADJACENT piece. ADJACENT keyword clarified. |
| **Excalibur's Gambit** | ADJACENT KNIGHTs +1 attack. Cluster strat. |

## B-Tier (Situational)

| Gambit | Why |
|--------|-----|
| **Templar's Gambit** ↓ nerfed to 1/3 | Former S-tier, now unreliable. Skip unless BISHOP-heavy. |
| **Family's Gambit** | PAWN wall +1 move. Solid but narrow. |
| **Scout's Gambit** | ADJACENT tile reveal. Good for exploration. |

## C-Tier (Niche)

| Gambit | Why |
|--------|-----|
| **Makibishi's Gambit** | Trap on ADJACENT. Still weak even after fix. |
| **Lich's Gambit** ↓ nerfed v1.2 | Never recovered from the v1.2 nerf. |
| **Squire's Gambit** | No Luck on KNIGHT move. Too situational. |
| **Mercenary's Gambit** | Enemy summon cost increase. Marginal. |
| **Super Hero Landing** | Landing pad tiles. Almost never optimal. |

## Removed in v1.3.0

- **Shield's Gambit** → replaced by Bubble Shield's
- **Trojan's Gambit** → replaced by Jockey's

## How the Meta Shifted

| Archetype | v1.1 Tier | v1.3 Tier | Why |
|-----------|-----------|-----------|-----|
| Templar-centric | S | B | 1/1 → 1/3 nerf |
| Protect Economy | B | A | Bunker's global +$1 |
| Knight Rush | B | A | Show Jumping + Whip both 1/1 |
| Lich builds | B | C | v1.2 nerf + no recovery |
| Blessed Tile | C | A | Graal's + blessed buffs |
| High-roll pivot | C | S | Missigno's enables pivots |
